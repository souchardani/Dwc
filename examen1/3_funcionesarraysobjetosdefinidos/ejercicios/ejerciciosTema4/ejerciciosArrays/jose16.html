Sintaxis: ObjetoArray.sort(sortFunction)
Nota: Si no usamos función de comparación para definir el orden, la matriz se ordenará siempre alfabéticamente. "500" vendrá antes que "7", pero en una ordenación numérica, 7 viene antes que 500. Este ejemplo muestra como usar la funcion de comparación que ordenará correctamente los elementos tanto en una matriz numérica como de cadenas.

Si se proporciona una función en el argumento sortFunction, debe devolver uno de los siguientes valores:

Un valor negativo si el primer argumento pasado es menor que el segundo.
Cero si los dos argumentos son equivalentes.
Un valor positivo si el primer argumento es mayor que el segundo.
